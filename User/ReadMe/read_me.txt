


日期2026.1.9
1.V4.0程序新增定时器2作为LED KEY USART等非阻塞式任务的时基，LCD刷新显示为阻塞任务放置在主循环中定时执行。
2.V4.0程序串口打印部分优化改进，由原先printf打印函数改为串口DMA发送，波特率提升至2000000。在VOFA中数据格式为JUST FLOAT。
3.V4.0程序改为ADC+DMA方式对母线电压和波轮电位器进行采样，解决了原先注入采样周期短，采样值偏低的问题。
4.V4.0程序增加了按钮和屏幕的交互方式，可以显示一些实时数据，可以通过按钮来切换电机的运行模式，具体如下：
                               长按KEY2（最上面的按键）  进入模式选择界面
							   长按KEY1（中间的按键）    退出模式选择界面
							   短按KEY2（最上面的按键）  向上选择模式
							   短按KEY1（中间的按键）    向下选择模式
							   注意！！！ 切换模式前需要将电位器逆时针扭到头
